# Чёрная Дыра: Поглощение

Веб-клиент: **Vite + React + TypeScript**, рендер **PixiJS v8**, состояние **Zustand**, тексты **react-i18next** (RU по умолчанию).

**Текущая версия:** см. `package.json` (`version`). До MVP — **`0.ФАЗА.патч`** (minor = фаза). Сейчас **0.3.0** — **фаза 2**: ветка A улучшений (радиус Шварцшильда, радиус притяжения, аккреционный диск и его визуализация), множитель цены улучшений **×1.45** за уровень; в симуляции — физика фазы 1 (`1/r²`, звезда, массы объектов); Obsidian под ТЗ v1.4–v1.5 и план UI (инспектор дыры, drawer улучшений, бета/MVP — см. `obsidian/11`). История версий: **`CHANGELOG.md`**. **1.0.0** — релиз MVP.

Перед **каждым коммитом**: поднять **patch** в `package.json` (`npm run version:patch`), кратко описать изменение в **`CHANGELOG.md`**, при необходимости обновить этот абзац и заметки в `obsidian/`. Правила номеров: `obsidian/10 — Версионирование релизов.md`, план фаз — `obsidian/11 — Дорожная карта до MVP (фазы).md`.

## Локально

```bash
npm install
npm run dev
```

Сборка:

```bash
npm run build
npm run preview
```

## Vercel

1. Импорт репозитория [github.com/Chuchumbrik/BlackHole](https://github.com/Chuchumbrik/BlackHole).
2. Framework Preset: **Vite** (или оставить автоопределение).
3. Команда сборки: `npm run build`, каталог вывода: `dist`.
4. Корень проекта — репозиторий; папка `obsidian/` не попадает в деплой (см. `.vercelignore`).

После деплоя проверьте главную страницу: canvas со сценой и React-панель поверх; вкладка «Улучшения» открывает покупку уровней.

## Документация по геймдизайну

Заметки для разработки лежат в каталоге `obsidian/` (хранилище Obsidian, в прод не включается). В именах файлов **нет символа `:`** (ограничение Windows); вики-ссылки в заметках приведены к фактическим именам.
