# Технические требования

Связано: [[00 — Главная (MOC)]], [[08 — Деплой — GitHub и Vercel]]

## Стек

- Vite + TypeScript + React
- PixiJS v8 (рендер мира)
- Zustand (состояние)
- react-i18next (RU + готовность EN)

## Архитектура

- React-оверлей поверх PixiJS canvas.
- Разделение слоёв: `simulation`, `economy/balance`, `world`, `ui`, `persistence`.
- Все формулы баланса должны иметь единый source-of-truth в коде.

## Симуляция

- Шаг по игровому `dt` (FPS-независимо).
- Упрощённая ньютоновская гравитация.
- Пересчёт траекторий в реальном времени при изменении параметров.

## Сохранения и оффлайн

- `localStorage`, автосохранение каждые 30 сек.
- Оффлайн-прогресс: 75%, максимум 12 часов.
- Пересчёт по последнему сохранённому состоянию (анти-эксплойт).

## UI и платформенность

- Fully responsive mobile + desktop.
- Touch-first для мобильных сценариев.
- Читаемость ключевых эффектов и предупреждений на узком экране.

## Лидерборд

- Процедурный топ-100 с seed от даты/штампа сохранения.
- Локальный лучший результат.
